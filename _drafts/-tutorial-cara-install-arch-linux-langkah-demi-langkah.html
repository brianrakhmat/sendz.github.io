Saya gak akan menjelaskan panjang lebar apa itu Arch Linux, tapi secara singkat, Arch Linux adalah distro linux. Lha iya, gak salah kan? Yasudah, post ini hanya untuk yang yakin dan siap install Arch, yang belum terbiasa dengan lingkungan text, mohon bersabar membaca tutorial ini.

Siapkan file iso archlinux

ISO file archlinux bisa diunduh di sini https://www.archlinux.org/download/, silakan memilih mirror paling dekat dari wilayah sekarang berada.

Buat bootable media archlinux

Ada beberapa cara untuk membuat bootable media, bisa burn ke kepingan CD / DVD, karena archlinux tidak lebih dari 700 MB, CD sudah cukup untuk menampung ISO archlinux. Karena bakar-bakar CD dan DVD sudah gampang, tinggal double-click file ISO lalu bakar dengan brasero, k3b dan semacamnya, di sini contohnya install arch dengan UFD (USB Flash Drive).

Siapkan Flash Drive, atau Flash Disk, koneksikan ke USB, lalu duplikasi file ISO ke UFD

$ sudo dd if=/lokasi_file_iso.iso of=/dev/sdX

sdX di sini adalah lokasi dimana UFD berada, perlu diingat bahwa sdX tidak boleh menggunakan angka contoh seperti sdc1, sdc2, cukup sdc saja.

Setelah selesai, boot komputer ke UFD, bisa dengan set boot priority di BIOS, saya menekan F12 ketika dialog BIOS muncul, kasus bisa berbeda jika komputer menggunakan UEFI.

Instalasi Arch

Koneksi Internet

Sebelum dimulai, sangat dianjurkan terkoneksi ke internet menggunakan ethernet, selain karena instalasi arch mengharuskan komputer terkoneksi ke internet untuk mengunduh paket yang diperlukan, koneksi internet melalui ethernet lebih mudah digunakan dibanding melalui wlan.

Setelah terkoneksi, coba ping ke google.

$ ping -c 3 www.google.com
